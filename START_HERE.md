## 🎯 7 API ENDPOINTS

| Endpoint | Method | Purpose |
|----------|--------|---------|
| `/api/messages` | GET | Get all messages |
| `/api/messages/stats` | GET | Get statistics |
| `/api/messages/:id` | GET | Get one message |
| `/api/messages` | POST | Create message |
| `/api/messages/:id` | PUT | Update message |
| `/api/messages/:id` | DELETE | Delete message |
| `/api/messages/broadcast/:id` | POST | Broadcast message |
